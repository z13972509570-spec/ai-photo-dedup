## 核心模块

### 1. Phash (core/phash.py)
- 多算法支持：aHash, pHash, dHash, whash
- 批量计算 + 多线程并行
- 相似度计算 + 重复检测

### 2. CLIP Features (core/clip_features.py)
- OpenAI CLIP 模型加载
- 批量特征提取
- 余弦相似度计算

### 3. Dedup Engine (core/dedup_engine.py)
- 双阶段检测流程
- 重复组聚类
- 最佳文件选择策略

### 4. Reports (reports/html_report.py)
- Jinja2 模板渲染
- 交互式 HTML 报告

## 数据流

```
扫描目录
    │
    ▼
┌─────────────────┐
│  阶段1: Phash   │  快速预筛 O(n²) → O(n log n)
│  感知哈希计算    │
└────────┬────────┘
         │ 疑似重复对
         ▼
┌─────────────────┐
│  阶段2: CLIP    │  深度语义验证
│  特征相似度      │
└────────┬────────┘
         │ 确认重复
         ▼
┌─────────────────┐
│  聚类分组        │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│  生成报告/清理   │
└─────────────────┘
```
